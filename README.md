# Network Deal — Site Web Premium

> Agence digitale créative | Tanger, Maroc

Site web ultra-moderne pour Network Deal — Next.js 16, TypeScript, Tailwind CSS v4.

## Démarrage rapide

```bash
npm install --legacy-peer-deps
npm run dev
# → http://localhost:3000
```

## Build & Déploiement

```bash
npm run build
npm start

# Vercel
vercel --prod
```

## Stack

- **Next.js 16.2.5** — App Router + Turbopack
- **TypeScript** — Typage strict
- **Tailwind CSS v4** — Styling utility-first
- **Framer Motion** — Animations premium
- **GSAP** — Scroll animations avancées

## Structure

```
src/
├── app/layout.tsx        # Layout + SEO + Schema.org
├── app/page.tsx          # Page principale
├── components/layout/    # Navbar, Footer, Cursor, Loader
├── components/sections/  # 11 sections premium
├── components/ui/        # Composants réutilisables
└── data/index.ts         # Toutes les données du site
```

## Personnalisation

Modifier les données dans `src/data/index.ts`
Modifier les couleurs dans `src/app/globals.css` (variables CSS)
